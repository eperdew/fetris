# fetris

Reimplementation of [TGM1](https://tetris.wiki/Tetris_The_Grand_Master) in Rust, running in the terminal.

## Build & Run

Build and run with cargo.

```sh
cargo run --release
```

Minimum terminal size: 37×22.

## Controls

| Key | Action |
|---|---|
| ←/→ or h/l | Move |
| ↓ or j | Soft drop |
| Space | Sonic drop |
| X | Rotate CW |
| Z | Rotate CCW |
| Q / Esc | Quit |
