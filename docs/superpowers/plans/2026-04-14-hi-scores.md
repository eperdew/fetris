# Hi Scores Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Persist per-mode/rotation hi score leaderboards to browser localStorage and display them on the Hi Scores menu screen.

**Architecture:** A new `hiscores` module owns all storage and ranking logic. `Judge` tracks the best grade and the tick it was first reached. At game end, `main.rs` submits one `HiScoreEntry` to the correct leaderboard. The menu's existing HiScores screen gains tabs and real data.

**Tech Stack:** Rust + macroquad, `quad-storage 0.1.3` (localStorage on WASM / file on native), `serde 1` + `serde_json 1`

### Task 2: Extend `Grade` with ordering and serialization derives

**Files:**
- Modify: `src/judge.rs`

`Grade` already derives `Clone, Copy`. We need `PartialEq, Eq, PartialOrd, Ord` (for ranking comparisons) and `Serialize, Deserialize` (for JSON storage). Because `#[derive(Ord)]` assigns ordinals in declaration order, and `Grade` is declared worst-first (`Nine` … `SNine`), the derived ordering correctly treats `SNine` as the largest value.

    entries.sort_by(|a, b| {

```rust
fn storage_key(mode: GameMode, rotation: Kind) -> &'static str {
    match (mode, rotation) {
        (GameMode::Master, Kind::Ars) => "hi_master_ars",
        (GameMode::Master, Kind::Srs) => "hi_master_srs",
        (GameMode::TwentyG, Kind::Ars) => "hi_20g_ars",
        (GameMode::TwentyG, Kind::Srs) => "hi_20g_srs",
    }
}

pub fn load(mode: GameMode, rotation: Kind) -> Vec<HiScoreEntry> {
    let storage = quad_storage::STORAGE.lock().unwrap();
    storage
        .get(storage_key(mode, rotation))
        .and_then(|s| serde_json::from_str(&s).ok())
        .unwrap_or_default()
}

pub fn save(mode: GameMode, rotation: Kind, entries: Vec<HiScoreEntry>) {
    let mut storage = quad_storage::STORAGE.lock().unwrap();
    if let Ok(json) = serde_json::to_string(&entries) {
        storage.set(storage_key(mode, rotation), &json);
    }
}
```

pub fn on_event(&mut self, event: &JudgeEvent) {
    match *event {
        JudgeEvent::LockedWithoutClear => self.combo = 1,
        JudgeEvent::ClearedLines {
            level,
            cleared_playfield,
            num_lines,
            frames_soft_drop_held,
            sonic_drop_rows,
            ticks_elapsed,
        } => {
            self.combo += 2 * num_lines - 2;
            let bravo = if cleared_playfield { 4 } else { 1 };
            self.score += ((level + 3) / 4 + frames_soft_drop_held + 2 * sonic_drop_rows)
                * num_lines
                * self.combo
                * bravo;
            let new_grade = Grade::of_score(self.score);
            if new_grade > self.best_grade {
                self.best_grade = new_grade;
                self.grade_ticks = ticks_elapsed;
            }
        }
    }
}
```

fn render_hi_scores(menu: &Menu) {
    const FONT: f32 = 22.0;
    const SMALL: f32 = 16.0;
    const LH: f32 = 28.0;

    let tab_names = ["MASTER / ARS", "MASTER / SRS", "20G / ARS", "20G / SRS"];
    let tab = menu.hi_scores_tab();
    let data = &menu.hi_scores_data()[tab];

    let cy = screen_height() / 2.0;

    // Tab header with left/right chevrons
    let label = format!("< {} >", tab_names[tab]);
    draw_centered(&label, cy - LH * 4.0, FONT, WHITE);

    // Entries
    for i in 0..5usize {
        let y = cy - LH * 2.5 + i as f32 * LH;
        let text = if let Some(entry) = data.get(i) {
            format!("{}. {:>2}   {}", i + 1, entry.grade.to_string(), format_time(entry.ticks))
        } else {
            format!("{}. ---", i + 1)
        };
        draw_centered(&text, y, SMALL, if i == 0 { WHITE } else { LIGHTGRAY });
    }

    draw_centered("BKSP to go back", cy + LH * 3.5, SMALL, GRAY);
}
```
